# Button

Tilburg button styling that cascades on top of `@utrecht/button-css`. Combine one size class (`tilburg-small | tilburg-medium | tilburg-large`) with one appearance modifier (`utrecht-button--primary-action | utrecht-button--secondary-action | utrecht-button--subtle`).

See the Tilburg storybook **Button** story for usage, appearance variants, and the "Vertical baseline shift" section that explains why text is optically nudged down to compensate for TradeGothicCondensed18's font metrics.
